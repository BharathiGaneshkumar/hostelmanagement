# Hostel Management System

A Java-based desktop application for managing hostel databases, built during undergrad at PSG College of Technology.

## What it does

Provides separate interfaces for admins and students to manage hostel operations — room allocation, student records, attendance tracking, and profile updates.

## Features

- Admin login and dashboard
- Student login and profile management
- Room allocation and availability tracking
- Attendance management
- View and update student records

## Tech Stack

- Java (Swing for UI)
- MySQL (via JDBC)
- IntelliJ IDEA

## Setup

1. Clone the repo
2. Add `mysql-connector-java-8.0.27.jar` to your project classpath
3. Set up a local MySQL database and update the connection credentials in the source files
4. Run the project from IntelliJ IDEA
